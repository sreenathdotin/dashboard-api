What NOT to do
Don't do:
# git reset --hard HEAD~3
because that moves the entire repository backward:
  help files
  other files
  commits
        Everything.



git restore --source=HEAD~3 main.py. 
    or for history HEAD~3 =4072441 , you can use , git restore --source=4072441 main.py
git add main.py
git commit -m "Restored main.py to version before print statements"
